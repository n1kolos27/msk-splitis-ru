# Инструкция по созданию Open Graph изображения

## Требования к og-image.jpg

Файл должен быть создан по пути: `assets/images/og-image.jpg`

### Технические характеристики:
- Размер: 1200x630 пикселей (соотношение 1.91:1)
- Формат: JPG
- Размер файла: < 200 КБ (рекомендуется 100-150 КБ)
- Качество: 85-90% (баланс между качеством и размером)

### Содержание изображения:
- Логотип компании или название "msk.splitis.ru"
- Текст: "Кондиционеры в Москве с установкой"
- Подзаголовок: "Официальная гарантия до 5 лет"
- Контактный телефон: "+7 (499) 757-57-19"
- Изображение кондиционера (опционально)
- Цветовая схема: соответствует брендингу сайта

### Альтернатива:
Можно создать несколько вариантов для разных разделов:
- `og-image-home.jpg` - для главной страницы
- `og-image-services.jpg` - для страниц услуг
- `og-image-catalog.jpg` - для страниц каталога
- `og-image-blog.jpg` - для статей блога

### Инструменты для создания:
- Adobe Photoshop
- Canva (https://canva.com)
- Figma
- GIMP (бесплатный)

### Проверка:
После создания проверьте изображение через:
- Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
- LinkedIn Post Inspector: https://www.linkedin.com/post-inspector/
