# Быстрая инструкция: HTTPS на localhost

## 3 простых шага:

### 1️⃣ Сгенерировать SSL сертификат
```bash
npm run cert:generate
```

### 2️⃣ Запустить HTTPS сервер
```bash
npm run server:https
```

### 3️⃣ Открыть в браузере
- **HTTPS:** https://localhost:5443
- Примите предупреждение браузера (это безопасно для localhost)

## Что проверить:

✅ Редирект HTTP → HTTPS работает  
✅ Страница 404 отображается корректно  
✅ Заголовки безопасности присутствуют  
✅ Нет ошибок смешанного контента
